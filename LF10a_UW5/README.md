# Mini-Shop UI Review (UW5)

Ziel: Heuristik-geführtes UI/UX-Review für einen kleinen Mini-Shop-Prototypen.

## Start
- Öffnen Sie `public/index.html` direkt im Browser.
- Oder: `npm install` und `npm run serve` (optional, wenn Node vorhanden ist).

## Aufgaben
- Prüfen Sie Sichtbarkeit von Zuständen und Feedback.
- Testen Sie Labels, Fokus, Tastaturbedienung, Modals und Fehlermeldungen.
- Dokumentieren Sie mindestens 5 konkrete Befunde im Review-Template.
- Setzen Sie 1–2 Quick Wins direkt am Prototyp um.

## Use-Cases
- UC1: Produktliste zeigt hilfreiche Statushinweise.
- UC2: Knöpfe und Icons sind verständlich und konsistent.
- UC3: Modal / Detailseite bleibt steuerbar per Tastatur.
- UC4: Fehlerfälle werden sichtbar und korrigierbar.
